# Ecom.tech test task

## Как запустить проект:

1. Склонируйте репозиторий:
```
git clone https://github.com/Birobizhan/ecom-test.git
```

2. Перейдите в папку проекта и установите необходимые пакеты:
```
cd ecom-test
npm install
```

3. Запустите проект:
```
npm run dev
```

## Структура проекта:

- src/App.jsx — основная логика приложения

- src/App.css — стили и адаптивная верстка

- src/mockData.js — имитация данных с сервера
